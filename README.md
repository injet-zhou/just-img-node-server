# just-img-node-server

just-img-node-server is the server of [just-img](https://github.com/injet-zhou/just-img)

## Get-started

### 1. Clone the repo

```bash
git clone https://github.com/injet-zhou/just-img-node-server.git
```
### 2. Install dependencies
```bash
npm i
# or
npm install
```
### 3. Setup your env
```env
PORT=YOUR_PORT
```

### 4. Run the app

```bash
node ./bin/www
```
